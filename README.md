# EdTech AI Learning Companion

> A modern, glass‑morphic dashboard for an AI‑powered learning companion.  
> Built with vanilla JavaScript, HTML, and CSS‑custom design tokens.

## 🎯 Project Overview

The **EdTech AI Learning Companion** is a single‑page web app that helps learners:

- Track progress on courses and hours studied.  
- Visualise a **progress‑over‑time** chart.  
- See recent activity and upcoming lessons at a glance.  
- Interact with a simple AI chat assistant for quick Q&A and quizzes.

All UI elements use a dark glass‑morphism style, vibrant accent colors, and smooth micro‑animations for a premium feel.
